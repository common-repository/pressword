== Description ==

Pressword enables an API interfaces between WordPress and the specificed API to be constructed and configured. This lets WordPress admin users communicate data to external APIs in reaction to WordPress events that they tap into.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
